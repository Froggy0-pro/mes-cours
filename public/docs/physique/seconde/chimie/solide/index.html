    Définition#

    Un solide ionique est un assemblage d’ions (cations + anions) organisés de manière régulière dans l’espace. Lorsqu’un solide ionique est placé dans de l’eau, il se dissout et se dissocie donc en ions dans la solution. Par exemple, le sel, ou chlorure de sodium, est un solide ionique de formule $NaCl$. Lorsqu’on met du sel (solide) dans l’eau, il se dissout en ions sodium $Na^+$ et chlorure $Cl^-$ selon l’équation:

    $$\ce{NaCl_{(s)} ->[\text{eau}] Na+_{(aq)} + Cl- _{(aq)}}$$

    • $(s)$ : solide
    • $(aq)$ : aqueux (mélangé dans l’eau)

    Exemples :

    • Chlorure de sodium : $NaCl$
    • Chlorure de calcium : $CaCl_2$
    • Sulfate de cuivre : $CuSO_4$

    Structure d’un solide ionique#

    Un solide ionique est constitué :

    • de cations (ions positifs)
    • d’anions (ions négatifs)

    L’ensemble est électriquement neutre.

    Exemple : $NaCl$

    • $Na^+$ : ion sodium
    • $Cl^-$ : ion chlorure

    Il y a autant de charges positives que négatives.

    Dissolution dans l’eau#

    Étapes de la dissolution#

    La dissolution d’un solide ionique se fait en plusieurs étapes :

    1. Rupture du réseau cristallin
      → les ions se séparent

    2. Hydratation des ions
      → les molécules d’eau entourent les ions

    3. Dispersion dans la solution
      → les ions se répartissent uniformément

    Concentration en solution#

    On définit la concentration molaire :

    $\boxed{C = \frac{n}{V}}$

    • $C$ : concentration (mol/L)
    • $n$ : quantité de matière (mol)
    • $V$ : volume (L)

    Concentrations effectives des ions#

    Lors de la dissolution, chaque ion a sa propre concentration.

    Exemple avec $CaCl_2$ :

    $CaCl_2 \rightarrow Ca^{2+} + 2Cl^-$

    Si on prépare une solution de concentration $C$ en $CaCl_2$ :

    • $[Ca^{2+}] = C$
    • $[Cl^-] = 2C$

    Exemple : Si $C = 0{,}10 , mol/L$ :

    • $[Ca^{2+}] = 0{,}10 , mol/L$
    • $[Cl^-] = 0{,}20 , mol/L$

    Nomenclature des solides ioniques#

    Un solide ionique se nomme :

    nom de l’anion + “de” + nom du cation

    Exemples :

    • $NaCl$ : chlorure de sodium
    • $CaCl_2$ : chlorure de calcium
    • $CuSO_4$ : sulfate de cuivre

    Cas particulier :
    Si le cation peut avoir plusieurs charges, on précise :

    • $FeCl_2$ : chlorure de fer(II)
    • $FeCl_3$ : chlorure de fer(III)

    À retenir#

    • Un solide ionique est un réseau d’ions
    • Il est globalement neutre
    • Dans l’eau, il se dissocie en ions
    • Chaque ion a une concentration propre
    • Le nom suit la structure : anion de cation

    1. $KBr_{(s)} \rightarrow K^+{(aq)} + Br^-{(aq)}$
    2. $[Al^{3+}] = C$ et $[Cl^-] = 3C$
    3. Sulfate de magnésium
